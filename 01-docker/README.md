# 01 - Docker

Learn containerization fundamentals, Dockerfile best practices, and docker-compose.

## Topics
- Dockerfile syntax and optimization
- Multi-stage builds
- Docker networking
- Docker volumes
- docker-compose for local development

## Getting Started
```bash
# Build an image
docker build -t myapp .

# Run with docker-compose
cd docker-compose
docker-compose up -d
```

## Exercises
1. Write a production-ready Dockerfile
2. Set up a multi-container application with docker-compose
3. Optimize image size using multi-stage builds
